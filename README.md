# DigitalCore Systems

Site institucional estático pronto para subir no GitHub e publicar facilmente.

## Estrutura
- `index.html`
- `styles.css`
- `assets/logo.png`

## Como usar
1. Extraia os arquivos.
2. Suba a pasta para um repositório no GitHub.
3. Para publicar rápido:
   - GitHub Pages
   - Netlify
   - Vercel
